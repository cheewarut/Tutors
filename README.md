ความสำคัญ

 	ได้รับความรู้ใหม่ๆนอกห้องเรียนโดยไม่มีค่าใช้จ่าย และเรียนเวลาไหนก็ได้ตามความสะดวกของผู้ใช้งาน

 	1.เพื่อศึกษาเกี่ยวการสร้างเว็บไซต์เพื่อใช้ประโยชน์ในด้านต่างๆ
